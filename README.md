# New Web App

This is the maintainable Next.js rebuild target for the Matin Consulting House site.

## Purpose

Recreate the frozen legacy website in a clean, source-first structure without changing the currently deployed site until parity is acceptable.

## Main folders

- `app/` - routes and root layout
- `components/` - reusable UI sections
- `content/` - structured page copy and data
- `public/images/` - migrated brand assets
- `styles/` - shared tokens and utilities

## Validation

- `npm run lint`
- `npm run build`
